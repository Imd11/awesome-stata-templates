# Changelog

All notable changes to this project will be documented in this file.

This project follows a lightweight release process. Versions summarize repository organization, documentation, template coverage, and maintenance workflow improvements.

## v0.1.0 - 2026-06-01

### Added

- Added project scope and maintainer roadmap.
- Added contribution guidelines for Stata templates.
- Added GitHub issue templates for bug reports, documentation improvements, and template requests.
- Added a pull request template with Stata-specific review checks.
- Added a file-by-file template index and method taxonomy.
- Added a maintenance audit documenting current repository strengths, gaps, and deferred work.
- Added lightweight repository checks for Markdown headings and local links.
- Added this changelog and release checklist.

### Changed

- Improved README navigation with quick links and clearer usage steps.
- Normalized Stata code fence language tags so GitHub syntax highlighting matches template content.

### Notes

- This release does not rewrite the Stata methods or change template semantics.
- Existing templates remain copy-paste oriented and should be adapted to each user's dataset and research design.
