# Generate various variables as required

stata generates a variable and randomly sets it to 1-30

```javascript
gen newvar = floor(30*uniform()) + 1
```

First use the uniform() function to generate a random number between 0 and 1, then multiply it by 30, then use the floor() function to round down, and finally add 1 to get a random integer between 1 and 30. Note: The random number generated by the uniform() function contains 0 but not 1, so we need to add 1 to ensure that the generated random integer is between 1 and 30.
